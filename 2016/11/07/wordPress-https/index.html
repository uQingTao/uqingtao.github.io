为 WordPress 全站开启 Https

Nov 07 2016

关于 WordPress 博客网站，也是刚接触不久，对于 Https 了解也是甚少。真正的了解也就是当时 Chrome 浏览器更新，我发现访问部分网站在域名前会有符号，后来了在网上搜了一下大致了解了 Https 相对于 Http 的优点，以及谷歌等都在推广 Https 。

后来在实现 Facebook 登录本站的时候，需要在开发平台注册，而注册的其中一项要求网站必须采用 Https 协议，所以就尝试实现。

申请证书

Https 首先需要向国际公认的证书证书认证机构（简称CA，Certificate Authority）申请 SSL 证书，申请证书的途径有很多，按照网站需求选择。

我选择的是腾讯云提供的免费 DV 证书，有效期为一年。

申请成功之后可以下载证书。解压获得3个文件夹，分别是Apache、IIS、Nginx 服务器的证书文件，按照服务器选择安装。

安装证书

按照腾讯云证书安装指引进行安装.本站运行在阿里云的服务器搭建的 LNMP(OneinStack) 环境，所以选择 Nginx 服务器的证书文件，上传到服务器 /usr/local/nginx/conf/vhost文件夹。修改 Nginx 配置文件：

在启用 Https 链接后，地址栏依然没有出现https。原来是网站引用了一些 HTTP resources ，找到所有原 Http 引用的静态资源及图片，改为 Https 。

又忙活了一大阵之后，刷新浏览器。终于出现，大功告成~
